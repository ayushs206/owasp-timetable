## OWASP Projects Site

This repository is organized to support multiple OWASP-related projects in a clear and scalable way.

### Project Structure

- The `/backend` folder is for all backend-related development and code changes.
- The `/websites` folder contains individual website projects.

### Adding a New Website Project

If you want to introduce a completely new site:

1. Create a new folder under `/websites`.
2. Start development inside that folder.
3. Read and follow that project's `README.md` standards to keep contributions consistent and easy for others to follow.

### Contributing to Existing Website Projects

If you want to add features to an existing site:

1. Go to the relevant project folder under `/websites`.
2. Implement your changes in that folder.

### Documentation Expectations

- Add or update the `README.md` whenever your changes require documentation updates.
